# 游戏资讯日报 · 验收标准

> 使用者：**主编**（审核每份交付物的尺子）。
> 用法：先查「通用合规项」，再逐条核对该阶段清单。**任一条不过 → 退回责任 Agent，写明改的方向和具体位置**（主编不代改），Agent 升版本重交。
> 来源：所有条目提炼自各角色《作业手册》的规格要求与自检清单，以及《游戏资讯日报 · 交付与流转规范》；本文件不新增要求。两边如有出入，以修订后最新一致版本为准。

---

## 0. 通用合规项（每份交付物都查）

- [ ] 文件名符合 `游戏资讯日报_责任方_日期_版本.zip`，责任方与版本正确（单一产出=Agent 名；多方合成=阶段名；派工单=主编）
- [ ] OSS 路径为 `csw/游戏资讯日报/{日期}/{阶段序号-阶段}/`，阶段序号正确（01-情报 / 02-选题 / 03-视觉 / 04-内容 / 05-成品 / 06-发布）
- [ ] 交付的是 OSS 返回的**完整 https 链接**（不能只写路径），zip 可下载、解压后以 `index.md` 为入口可读
- [ ] `index.md` 元信息头字段齐全：任务 / 类型 / 交付 Agent / 阶段 / 版本 / 时间 / 上游来源 / 状态 / 自检
- [ ] `上游来源` 写「来源方 · 完整 https 链接」，多个上游逐条列
- [ ] 附件按相对链接引用且全部可打开（如 `images/`、`source/`、`html/`）
- [ ] 返工件版本号已递增（v2/v3…），原版留底未删

---

## 01-情报（游戏情报员）

**内容质量**
- [ ] 官方情报为**完整内容**（完整文字 + 图片），来源如官方账号、官方 subreddit、厂商发布
- [ ] 社区玩家讨论为**热门内容**（热门帖 / 高赞讨论的文字 + 图片）
- [ ] 每条均标注来源平台（Reddit / X）、原文链接、发布时间，且原文链接可达
- [ ] 已去重；无与游戏无关或无法溯源的内容
- [ ] 条数够当日所需

**规范合规**
- [ ] `index.md` 分「官方情报 / 社区玩家讨论」两栏
- [ ] 图片已下载进 `images/`（非外链裸 URL），正文相对链接可用
- [ ] 自检申报：全程只用 opencli reddit / twitter 采集，并按规定插入随机间隔

## 02-选题（选题研究员）

**内容质量**
- [ ] **文章主标题 ≤ 64 字符**（发布员将用作公众号文章标题）
- [ ] **封面标题最多两行、每行 ≤ 5 个字**（视觉设计师将放到封面）
- [ ] 摘要齐全，契合公众号调性；**字数 ≤ 120 字符**
- [ ] 每个选题点能对应回资讯包具体条目

**规范合规**
- [ ] `index.md` 含四部分：文章主标题 / 封面标题 / 摘要 / 对应情报

## 03-视觉（视觉设计师）

**内容质量**
- [ ] 比例 **2.35:1**；尺寸 **1920 × 817 px**（如附小尺寸版，为 720 × 306 px）
- [ ] 用户素材背景未改：人物、场景、天空、色彩、光效保持原样；未重绘角色、未改人物动作与背景构图
- [ ] 左侧角色区：脸部、武器、主要动作区域未被遮挡
- [ ] Logo 与标题**全部在右侧 1:1 标题安全区内**，未越界到左侧角色区
- [ ] Logo 位于安全区上方、水平居中、清晰可见，未重设计 / 扭曲 / 遮挡
- [ ] 标题用的是选题研究员的**「封面标题」**（≤2 行 / 每行 ≤5 字），位于 Logo 下方，Logo 与文字互相居中对齐，字号大、远看可读
- [ ] 字体厚重、醒目、方正，在素材背景上可读性强；整体明亮简约，像官方封面而非普通海报
- [ ] 无日期、平台、作者名、水印等多余文字

**规范合规**
- [ ] 封面图在 `images/`；`index.md` 含设计意见（所用素材 / 提示词、思路、可调整项）
- [ ] 自检申报：封面由 `generate-images` skill 生成

## 04-内容（深度内容作者）

**内容质量**
- [ ] 条目涵盖主要资讯，与选题方向一致
- [ ] 新游戏发布条目含**发售时间 / 平台 / 港区价格**
- [ ] 事实与资讯包核对一致；关键信息出处可溯回情报原文链接
- [ ] 配图与正文内容对应

**规范合规**
- [ ] 正文为 md 格式；**不含文章主标题和摘要**（公众号平台字段，由发布员填入）
- [ ] 配图在 `images/`，正文相对链接可用
- [ ] 末尾含页脚两行：① `数据来源：` 已按当日实际来源填写（每日不同）；② 「整理：Lemon，「每日花生新闻」将持续带来最新游戏资讯与动态，欢迎点赞、关注与分享支持。」逐字照抄，未改动

## 05-成品（主编合成前自检）

- [ ] 视觉、内容两份均为**已过用户审**的最新版本
- [ ] `上游来源` 两条链接（视觉 / 内容）齐全无误、可下载
- [ ] 责任方写「成品」，类型=成品；含主编意见与下一步安排

## 06-发布（发布员）

**内容质量**
- [ ] 正文 HTML 已生成、排版正常
- [ ] HTML 内图片已用 `mp-helper` 上传到公众号素材库，并已替换为素材库地址（无外链残留）
- [ ] 草稿箱字段齐全：文章封面图已设置；**标题=文章主标题（≤64 字符）**；摘要已填；**作者=`CCS_Lemon`**
- [ ] 已保存进微信公众号**草稿箱**

**规范合规**
- [ ] 正文中已插入两个广告组件：第一个广告前正文 >300 字；两广告之间正文 >300 字；位置在模块最后一段 `<p>` 后、`</section>` 前；广告代码属性未改动
- [ ] 交付包含 `html/article.html`（图片替换后的公众号正文 HTML）；`index.md` 写明草稿箱位置与已填字段核对
- [ ] 发布员**先通知主编**（而非用户）
